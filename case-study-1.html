Predictive Patient Readmission Model

Problem Statement

Hospital readmissions within 30 days are costly and often preventable. Identifying at-risk patients can optimize resource allocation and improve care.

Methodology

Used a logistic regression model trained on EHR data, including demographics, diagnoses, and prior admissions. Applied feature engineering to create risk scores and used cross-validation to ensure robustness.

Tech Stack

  • Python (Pandas, Scikit-learn)
  • Jupyter Notebook
  • SQL for data extraction

Impact

Reduced readmission rates by 10% in a pilot study, saving an estimated $500,000 annually for the hospital.
